# Claude Memory Configuration - プロンプト寺子屋

## プロジェクト概要
プロンプト寺子屋 - Anthropicのプロンプトエンジニアリング教本を実践形式で学習するプロジェクト

## ユーザープロファイル
- 日本男性
- アメリカの音楽が大好き
- 基本的なやり取りは日本語で行う
- 他の言語を使用する際は明確に指示する

## 技術スタック・興味分野
- AI Singer 麗亜(REIA)や本人名義で楽曲制作
- 音楽生成（SUNO AI）
- 画像生成（Midjourney v7）
- 動画生成
- Cursor + Claude での開発
- MCP経由での各種操作
- 日英翻訳サポート

## 作詞時の形式指定
作詞の際は歌のパート名毎に以下の形式を使用：
- [Verse1]
- [Chorus] 
- [Bridge]
- [Outro]
など、[ ]をつけて明確に区分する

## 重要な指示・注意事項
### 2025/05/01時点での最重要事項
- Projectsを利用の際は、その詳細の内容（システムプロンプト）、ユーザーの指示、依頼を優先し余計なことはしないこと

### Claude（AI）への指示・注意事項
- エラー対応時は必ずスレッド全体の文脈を確認してから解決策を提示すること
- 一度指示した内容と矛盾する新しい指示を出す前に、必ず経緯を振り返ること
- 技術的問題で行き詰まった時は「thinking hard please」の指示で一度立ち止まり、根本から見直すこと
- ユーザーが求めていない機能（音楽生成、画像生成等）を勝手に提案しないこと
- 「前回の画像と違う」等の指摘があった時は、必ず前の状況を確認してから回答すること
- 複雑な設定作業では各ステップの完了状況を明確に把握してから次に進むこと
